# Alfido Tech Data Analytics Internship

## Overview

This repository contains the projects completed during the **Alfido Tech Data Analytics Internship**. The internship focused on developing practical skills in data cleaning, exploratory data analysis (EDA), data visualization, dashboard development, and business reporting using real-world datasets.

All projects were completed using **Microsoft Power BI** with interactive dashboards, key performance indicators (KPIs), business insights, and actionable recommendations.

---

## Tools & Technologies

- Microsoft Power BI
- Power Query
- DAX (Data Analysis Expressions)
- CSV Datasets
- Microsoft Excel

---

# Project 1: Customer Behavior Analysis

### Objective

Analyze customer purchasing behavior to identify buying patterns, customer segments, and product preferences.

### Features

- Data Cleaning
- Customer Segmentation
- Product Category Analysis
- Payment Method Analysis
- Purchase Trend Analysis
- Returns Analysis
- Interactive Dashboard

### KPIs

- Total Revenue
- Total Customers
- Total Returns
- Average Purchase

### Key Insights

- Identified high-value customers.
- Compared product category performance.
- Analyzed payment preferences.
- Studied customer age groups and purchase trends.

---

# Project 2: Sales Performance Analysis

### Objective

Analyze sales performance across categories, regions, customer segments, and products to support business decision-making.

### Features

- Sales Trend Analysis
- Regional Performance
- Category & Sub-Category Analysis
- Customer Segment Analysis
- Top Products Analysis
- Interactive Dashboard

### KPIs

- Total Sales
- Total Orders
- Total Customers
- Average Sales

### Key Insights

- Technology category generated the highest sales.
- West region showed the best performance.
- Consumer segment contributed the highest revenue.
- Sales showed consistent growth over time.

---

# Project 3: HR Analytics – Employee Attrition Analysis

### Objective

Analyze employee data to identify attrition patterns and provide insights for improving employee retention.

### Features

- Attrition Analysis
- Department-wise Employee Distribution
- Job Role Analysis
- Gender Analysis
- Work-Life Balance Analysis
- Monthly Income Analysis
- Interactive Dashboard

### KPIs

- Total Employees
- Employees Left
- Active Employees
- Attrition Rate
- Average Monthly Income
- Average Age

### Key Insights

- Identified departments with higher attrition.
- Compared attrition across job roles.
- Analyzed work-life balance and employee demographics.
- Provided recommendations to improve retention.
